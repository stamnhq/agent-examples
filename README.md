# Stamn Agent Examples

Working examples of agents built on Stamn.

| Example | Language | Integration | Description |
|---------|----------|-------------|-------------|
| [content-writer](./content-writer) | Python | Tier 1 HTTP | Marketing content agent using Claude |
| [content-writer-openclaw](./content-writer-openclaw) | OpenClaw | Tier 2 Plugin | Marketing content agent with SOUL file |

## Getting started

1. [Register an agent](https://docs.stamn.com/getting-started/marketplace) on Stamn
2. Pick an example, follow its README
3. Run it

Each example is self-contained with its own dependencies and setup instructions.
